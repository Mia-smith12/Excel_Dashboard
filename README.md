# Restaurant Performance & Guest Experience Dashboard
A fully interactive Excel dashboard analyzing sales, guest satisfaction, and operational performance across multiple restaurant locations.

# Project Overview
This project is an interactive Excel dashboard designed to help restaurant managers and stakeholders monitor business performance and guest experience metrics in one place.
It provides a clear view of sales trends, customer satisfaction, menu performance, and order behavior across locations.

The dashboard is built using:
* Pivot tables
* Slicers
* Data modeling
* Custom UI design

# The Data
The dataset includes restaurant transaction and guest feedback data. Key fields include:
* Order date
* Location
* Menu item
* Category (Main, Beverage, Appetizer, Dessert)
* Order type (Dine‑In, Delivery)
* Sales amount
* Customer rating (1–5)
Data has been cleaned and structured for pivot‑table analysis.

# Dahboard Use Cases 
The dashboard aims to:
* Track total and monthly sales
* Monitor guest satisfaction through average ratings
* Compare ratings by location
* Identify top‑selling menu items
* Analyze category performance (mains, beverages, appetizers, desserts)
* Measure total orders and order‑type distribution
* Provide a clean, interactive tool for decision‑making

# The Process
Steps taken to build the dashboard:
* Cleaned and standardized raw data
* Created pivot tables 
* Built calculated fields for KPIs
* Designed slicers for location, category, & order type
* Applied custom slicer formatting for a modern UI
* Assembled all visuals into a single dashboard layout

# Dashboard Features
The dashboard includes:
* KPI cards: Total Sales, Total Orders, Avg Rating
* Monthly sales trend chart
* Ratings by location bar chart
* Top menu items table
* Category performance breakdown
* Order type comparison (Delivery vs Dine‑In vs Takeaway)
* Interactive slicers for filtering by location, category, and order type

# How to Use
* Use slicers to filter by location, category, or order type
* Hover over charts for detailed values
* Refresh pivots if new data is added
* Unhide backend sheets if you want to inspect the data model
